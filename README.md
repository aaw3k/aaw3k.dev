# aaw3k.dev

Personal site built with Next.js. MDX, Framer Motion] and styled with CSS Modules.

## Run Locally
```bash
git clone https://github.com/aaw3k/aaw3k.dev
cd aaw3k.dev
pnpm
pnpm dev
```
